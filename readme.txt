# 🌍 AI Travel Planner

Plan smarter trips using AI! This app uses **Google Gemini**, **CrewAI**, and **Streamlit** to create budget-friendly, historical, and weather-aware travel plans.

---

## ✈️ What It Does

You give it:
- A **destination**
- A **budget**

It gives you:
- 🔎 Historical sites + real-time weather
- 💸 Budget flights, hotels, food, etc.
- 🗺️ A full 3-day itinerary

---

- 🤖 [CrewAI](https://docs.crewai.com/) — Multi-agent task delegation
